# Отчёт к лабораторной работе №4. B-дерево

## Задача
Реализовать B-дерево порядка 2 с операциями вставки, удаления, поиска и печати.

## Вариант
N=16, (16%3)+1 = 2 — B-дерево порядка 2 (2-3-4 дерево).

## Реализованные функции
- `createNode` — создание узла
- `findNode` — поиск по ключу
- `splitChild` — расщепление узла
- `insertRecursive` / `insert` — вставка
- `borrowFromLeft` / `borrowFromRight` — заимствование
- `merge` — слияние узлов
- `fill` — пополнение узла перед удалением
- `deleteRecursive` / `delete` — удаление
- `printTree` — печать дерева
- `processFile` — обработка команд из файла

## Тесты (GTest)
- Создание узла
- Вставка одного элемента
- Вставка трёх элементов (сохраняется порядок)
- Поиск существующего и отсутствующего ключа
- Удаление из листа
